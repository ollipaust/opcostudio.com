# www.opcostudio.com gatsby frontend

## Features

* based on:
  * React
  * Gatsby
  * styled-components
  * three.js

## Installation

* `git clone https://github.com/ollipaust/opcostudio.com.git opcostudio`
* `cd opcostudio`
* `yarn install`

# Development
yarn develop

# Build
yarn build

# Full Build
yarn ssr

# Fast Build (no icons etc)
yarn prod 

# Prettier formatting
yarn format

# Generate Icons
yarn icons

# Lighthouse Test
yarn test
